# Version

The `amp version` command displays the current version of AMP.

### Usage

    $ amp version

    Client:
     Version:       v0.10.0-dev
     Build:         6f590348
     Server:        cloud.appcelerator.io:50101
     Go version:    go1.8
     OS/Arch:       darwin/amd64

    Server:
     Version:       v0.10.0-dev
     Build:         04daab98
     Go version:    go1.8.1
     OS/Arch:       linux/amd64
