# version

The `amp version` command displays the current version of AMP.

### Usage

```
    $ amp version

    amp:
     Version:       v0.6.0-dev
     Build:         16cb488f
     Address:       172.17.0.1:50101
     Go version:    go1.8
     OS/Arch:       linux/amd64

    amplifier:
     Version:       v0.6.0-dev
     Build:         f11025f6
     Go version:    go1.8
     OS/Arch:       linux/amd64
```
